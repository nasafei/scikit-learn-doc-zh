# scikit-learn(sklearn) 官方文档-中文版
![](doc/en/logos/scikit-learn-logo.png) 

官网地址: [scikit-learn(sklearn)](http://scikit-learn.org): <http://scikit-learn.org>

scikit-learn 是基于Python语言的机器学习工具。
1. 简单高效的数据挖掘和数据分析工具
2. 可供大家在各种环境中重复使用
3. 建立在NumPy，SciPy和matplotlib上
4. 开源，可商业使用 - BSD许可证
